# Grilling — Adversarial Decision-Tree Questioning

This mode stress-tests thinking through relentless questioning about plans, decisions, or ideas.

## Key Approach

- Walk down each branch of the decision tree, resolving dependencies between decisions one-by-one
- Pose questions individually, awaiting the response before advancing
- For each query, provide a recommended answer
- Look up factual information via available context rather than asking when the answer is knowable

## Critical Constraint

Act only after confirming mutual understanding. Never ask multiple questions simultaneously — this creates confusion. One question, one answer, then the next.

## Scope Distinction

Explore context-accessible facts independently, but present all judgment-based decisions to the user for their determination. The goal is to surface every unresolved fork in a plan/design before implementation begins, not to grill for its own sake.

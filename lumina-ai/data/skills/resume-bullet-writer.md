# Resume Bullet Writer

Most resumes list *duties* ("Responsible for managing team") instead of *achievements*. Every bullet should show impact with a number.

## Frameworks

**X-Y-Z (Google):** "Accomplished **[X]** as measured by **[Y]** by doing **[Z]**."
> "Grew Instagram following 250% (5K→17.5K) by running a daily content calendar and influencer partnerships."

**STAR (condensed to one line):** Situation → Task → Action → **Result with metric**.
> "Revitalized an underperforming sales team via a new training program and commission restructure, lifting quota attainment 65%→92% and generating $1.8M added revenue."

**CAR:** Challenge → Action → Result — a tighter alternative to STAR.

## Strong Verbs (never "responsible for" / "helped" / "assisted")

Lead/Directed/Spearheaded · Achieved/Delivered/Exceeded · Grew/Increased/Scaled/Doubled · Built/Launched/Pioneered · Streamlined/Optimized/Automated · Analyzed/Diagnosed/Forecasted · Resolved/Reduced/Eliminated.

## Quantify Every Bullet

Include at least one number — money ($2M revenue, $50K saved), percentage (45% conversion lift), time (8s→2s load), scale (team of 15, 100K DAU), or before→after.

**When you lack exact numbers:** approximate with `~`, use ranges, use conservative estimates (thought 60%? say 50%), or quantify *inputs* instead of outputs ("conducted 30+ customer interviews", "analyzed 500+ data points").

## Common Mistakes

Passive language → active verbs · no metric → add a number · duty not achievement → show the result · vague → specific scale · too long → keep to 1–2 lines.

## Process

Start from the weak bullet → ask what/how-much/what-actions/what-impact → gather the facts → apply X-Y-Z or STAR → trim to 1–2 lines, keeping the most impressive metric.

**Strength checklist per bullet:** strong verb ✓ · ≥1 metric ✓ · concrete result ✓ · scale/context ✓ · ≤2 lines ✓ · reads as achievement not duty ✓ · relevant to the target role ✓.

For **entry-level/career-changers**, mine academic projects, internships, volunteer leadership, and transferable skills reframed for the target industry — with numbers.
